The Firebase API is made available through a JS library. The global “firebase” object provides access to the database, storage, auth, initialization and more.

The initialization script is generated for you within the Firebase Console and available under the “Add Firebase to your web app” link on the Project Overview page.

The Database object can provide a reference to a specific path by providing the full path as a parameter to .ref() however by default the reference is created to the root of the database, or slash (/).

You can use any number of .child() method calls on the reference to drill down into a specific object, and can even select an object attribute using this technique.
