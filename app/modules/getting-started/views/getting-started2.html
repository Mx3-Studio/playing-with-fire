Click Database

Click the ellipsis (three dots) and select Import JSON. Select the "starter-database.json" file from your Fireside project folder and click Import.

Open the new Fireside project

Click Database then Rules

For now change the rules to

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

Then click Publish

Note that you NEVER want to leave your permissions unrestricted like this, but we will fix this later in the workshop.

You’ll also need to prepare your database by loading the sample data provided and removing all security restrictions from your database. (The database instructions are on the next page on the website)

NOTE: You don’t ever want to leave a database in this unrestricted state. We are doing this here so that we can introduce Firebase to you one step at a time. We will be correcting this security deficiency later in the workshop.
