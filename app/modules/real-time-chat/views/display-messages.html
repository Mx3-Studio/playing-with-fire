Instructions

Update the listChats method in the Chats service to get the chat thread from /chats in the database (TODO [RTC-1])

Hints

Use the firebase Reference to select objects by path using child(): firebase.database.Reference

Use the AngularFire $firebaseArray for chats: $firebasearray

QA

Now run the application and visit http://localhost:8080/#/chats

You should see all of the demo chats show up.

Chats contain the following attributes:

  • content: the chat message
  • timestamp: the UTC string representing the moment the chat was created

Javascript

// reference to database root: /
var db = firebase.database().ref();

// reference to chats
var chatsRef = db.child('/chats');

// read
chatsRef.child('-Kf9kfiG0rwnlB1gtquA')
.once('value').then(function(snapshot) {
  var message = snapshot.val().content;
});

// read
var data = $firebaseArray(chatsRef);
