Firebase is a Google Platform as a Service (PaaS) which makes writing and deploying web and mobile applications faster and easier than ever. Now a front-end web developer can create fully-featured applications without needing to manage servers or infrastructure. In this workshop you will learn how easy it can be to take your app idea from concept to production, using the fully-featured Firebase platform.

Some of the key Firebase features include:

  • Web, Android, iOS support (as well as other languages)
  • Cloud-hosted object database
  • Real-time database (can easily achieve 3-way binding from Browser forms to Database and back)
  • OAuth support - no more painful implementations, can add OAuth to your app in minutes
  • Includes hosting for your static web application including CLI deploy
  • Manage your entire Firebase account with their web console, CLI client, or REST API
  • Complex security rules to provide object-level control over read/write/validation of your data
  • Framework-specific APIs including AngularFire, ReactFire, and BackboneFire
  • File storage w/ security rules
