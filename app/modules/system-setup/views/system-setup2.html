Open your console / command shell and navigate to your workspace folder. For me this is Bash followed by

$ cd ~/workspace

Checkout the project

$ git clone https://github.com/Mx3-Studio/fireside.git
$ cd fireside
$ npm install
$ bower install

Test the project

$ gulp validate
$ gulp serve-dev

Preview the application - http://localhost:8080/
