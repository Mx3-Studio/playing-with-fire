Mac

You’ll need Homebrew, Node, & NPM, and should probably have Gulp and Bower installed globally. You’ll need to be familiar with your command prompt at this point.

Check to see if these are installed:

# Homebrew
$ brew -v

# Git
$ git --version

# Node & npm
$ node -v
$ npm -v

# Gulp
$ gulp -v

# Bower
$ bower -v

To install everything run the following

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew install git
$ git config --global user.name "My Name"
$ git config --global user.email "myemail@gmail.com"
$ brew install node
$ npm install -g gulp bower

Linux

You’ll need Node & NPM, and should probably have Gulp and Bower installed globally. You’ll need to be familiar with your command prompt at this point.

Check to see if these are installed

# Git
$ git --version

# Node & npm
$ node -v
$ npm -v

# Gulp
$ gulp -v

# Bower
$ bower -v

To install git, node and npm run the following

Debian / Ubuntu / Mint

$ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
$ sudo apt-get install -y nodejs
$ sudo apt-get install -y build-essential nodejs-legacy git-all

Fedora

$ yum install nodejs npm git-all

Others

  • Installing Node.js via package manager
  • Git: Download for Linux and Unix

Then (everyone)

# Configure Git
$ git config --global user.name "My Name"
$ git config --global user.email "myemail@gmail.com"

# Install NPM packages
$ npm install -g gulp bower

Windows

You’ll need Node & NPM, and should probably have Gulp and Bower installed globally.

  1. Run git installer
  2. Run node installer

At this point you’ll need to open up Git Bash (or another bash client) to complete the following commands

# Configure Git:
$ git config --global user.name "My Name"
$ git config --global user.email "myemail@gmail.com"

# Install the NPM packages:
$ npm install -g gulp bower
