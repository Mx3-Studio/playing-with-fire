Getting Started: Configure Firebase connectivity

Click “Overview” then click the “Add Firebase to your web app” icon to view the configuration script.

Copy the configuration script,

then open up “env/dev/config.js” in your Fireside project and replace the placeholder values in this script with the ones you copied out of the Firebase Console.

Save the file, then run the project

$ gulp serve-dev

http://localhost:8080/#/check

If the script comes back with a confirmation page then you are successfully connected to Firebase. If not, then some debugging will be in order.
