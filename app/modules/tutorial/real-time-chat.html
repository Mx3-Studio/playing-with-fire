Real-Time Chat

Success Criteria

  • Display chat messages Firebase
  • Create new chat messages
  • Click on Hashtags to see only related chats

Instructions

  1. Update the ChatController to get the chat thread from the database - /chats
  2. Modify the createChat function in ChatController to get a key for the new chat object - /chats/{chatId}
  3. Modify the createChat function to push the new chat to the chat thread by adding a path reference to the updates - /chats/{chatId}

Hints

  • Use the firebase Reference to select objects by path using child(): firebase.database.Reference
  • Use the AngularFire $firebaseArray for chats: $firebasearray
  • Use the push method on the Reference to push a new object to Firebase and obtain the unique key
  • Update multiple items at once: Updating or deleting data
  • Chats contain the following attributes:
    • content: the chat message
    • timestamp: the UTC string representing the moment the chat was created

QA

Now run the application and visit http://localhost:8080/chats

You should see all of the demo chats show up.
You should be able to submit new chats which show up in the chat stream immediately and automatically.
