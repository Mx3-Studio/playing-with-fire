Getting Started: Setup the Firebase Console Account

Login to https://console.firebase.google.com using your Google account.

Click Create New Project

...and give it the name "Fireside"

Open the new Fireside project

Click Database then Rules

For now change the rules to

  {
    “rules”: {
      “.read”: true,
      “.write”: true
    }
  }

Then click Publish

Note that you NEVER want to leave your permissions unrestricted like this, but we will fix this later in the workshop.
