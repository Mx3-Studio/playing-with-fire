Challenges: Deploy with Cordova

Success Criteria

  • Run in Android emulator using Cordova

Instructions

  1. Install cordova and create a new project:
    1. $ npm install -g cordova
    2. $ cordova create fireside
  2. Publish the Fireside build to Cordova:
    1. $ rm -rf fireside/www
    2. $ gulp build
    3. $ mv public fireside/www
  3. Add the browser platform and run:
    1. $ cordova platform add browser
    2. $ cordova run browser
  4. Install the Android SDK and the latest API version
  5. Add the android platform and run:
    1. $ cordova platform add android
    2. Install Firebase Android Auth plugin (TBD)
    3. $ cordova run android

Hints

  • Apache Cordova - Apache Cordova
  • Installing Android SDK - Installing Android sdk Windows Mac and Linux Tutorial
  • Auth plugin - Cordova Plugin Firebase Auth

QA

Test the app by running in the browser, in the Android emulator, and if possible on your own Android device!

If you have an Android phone you can test it out by connecting via USB and running:

$ cordova run android --device
