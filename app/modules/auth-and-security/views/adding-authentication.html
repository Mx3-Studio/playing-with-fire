Success Criteria

Login using Google OAuth

Instructions

Enable the Google Sign-in Provider from Firebase console

Implement methods in the FirebaseAuth service:

  • authenticate()
  • logout()

Hints

You will use Firebase Authentication to perform Google OAuth: Firebase Authentication

Use AngularFire $authWithOAuthPopup to login using Google: Angularfire Users and Authentication: $authWithOAuthPopup(provider[, options])

QA

Now run the application and visit http://localhost:8080/#/login

You should now be able to click the Google Login button to authenticate using your Google account (the same one you used to sign up for Firebase would be fine).

// Firebase API OAuth Login
firebase.auth()
.signInWithPopup('google')
.then(function(userCredential) {
  console.log(userCredential);
});

// AngularFire OAuth Login
$firebaseAuth()
.$signInWithPopup('google')
.then(function(user) {
  console.log(user);
});

// AngularFire Logout
$firebaseAuth().$signOut();
