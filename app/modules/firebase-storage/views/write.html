Files / Blobs

angular-file-model (https://github.com/ghostbar/angular-file-model)

// Create a root reference
var ref = firebase.storage().ref();
//	Files & Blobs
ref.child('images').put(file)
.then(function(snapshot) {
  console.log('Uploaded a blob or file');
});

MAM Say: You can store files, blobs, base64 and more using the put() method to upload them to your storage bucket.

We are using angular-file-model which is a Simple Angular.js directive that allows input element's of type file to be handled in models inside a controller's scope.

You can also manage uploads by pausing, resuming, or canceling.

Notes: Then add thestorageBucket attribute to your config

Upload from:

  • Blob
  • File
  • Byte Array
  • String
    • Raw string
    • Base64
    • Base64url
    • Data Url

Can also manage uploads:

  • Metadata
  • Pause
  • Resume
  • Cancel

And Monitor progress
