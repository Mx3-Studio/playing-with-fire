Metadata includes

  • Name
  • Size
  • content Type (MIME Type)
  • fullPath
  • downloadURLs

// Create a reference to the file whose
// metadata we want to retrieve
var ref = firebase.storage().ref()
  .child('images/myMeme.png');

// Get metadata properties
ref.getMetadata()
.then(function(metadata) {
  // Metadata now contains the metadata
  // for 'images/myMeme.png'
}).catch(function(error) {
  // Uh-oh, an error occurred!
});

MAM Say: Once you have a reference to your Firebase storage bucket you are able to retrieve the metadata including the download url from a storage location and filename.

The example here shows how you can access the file myMeme.png and then do something with it.
